# 🌀 Cuaderno Interactivo: Movimiento Oscilatorio

Bienvenido al repositorio oficial del **Cuaderno Interactivo sobre Movimiento Oscilatorio**, un recurso educativo diseñado para explorar los principios fundamentales de la física mediante simulaciones y teoría aplicada.

## 👥 Autoría y Créditos
Este proyecto es una obra original desarrollada íntegramente por:
* **Oscar Alejandro Ramírez Espinosa**
* **Jeny Maritza Mora Cardona**

El cuaderno ha sido diseñado y estructurado utilizando la plataforma **Antigravity**, integrando lógica de programación y contenido pedagógico propio.
